CLASSICAL TABS - WANTS LIST - PLEASE HELP!

how to find tabs -

many of the requests that come in are for pieces which are already available online, sometimes on the main page of this site! so first make sure the piece you're looking for isn't on the Classical Guitar Tablature page, then try searching the web, the archives for the newsgroup alt.guitar.tab and rec.music.makers.guitar.tablature, the main tab sites and their associated archives Power Tab, TuxGuitar, TablEdit, Classclef, GProTab etc), and (if you can face the ads) Ultimate Guitar -- there are some specialist classical tabs sites listed on the Classtab links page -- there are also tab files available on the various file-sharing networks (eMule, The Pirate Bay, Demonoid etc).

if the tab can not be found and the request is for an acoustic fingerstyle classical composition, then you are welcome to email me asking for it to be added to the wants list below -- you could also try using one of the MIDI to tab conversion programs such as Power Tab -- the fingering may be wrong, but it should gives you the correct notes -- (fingering can be corrected from the sheet music or by watching YouTube vids) -- if you need a tab done urgently, use one of the online music engraving or custom transcription services


i regularly get emails from people asking me to do tabs for them -- these are added to my "tabs to be done" list, and are generally dealt with on a FIFO principle (First In, First Out) -- some of the backlog has now been cleared and i'm currently up to August 2002 ("Tears from the Sky" by Demetis Dionisis) -- (please note: i'm no longer young, and it's unlikely that i'll be starting on requests from the current decade until well into my next reincarnation)

if you need sheet music or tabs in various formats and you're not in a hurry, and you're confortable with p2p networks, then check out mbwanna's 20GB excellently indexed collection of classical guitar sheet, tabs & MIDIs (eMule ed2k link) -- Note: if you have no experience of filesharing, then take care! a 360K '.exe' file with 600 seeds and 50 alternative titles which claims to contain the complete scores of Beethoven's Choral and Symphonic works is unlikely to do so -- scan all downloads for malware and viruses)

and if you're about to write in to tell me that you can't find the tab of a piece that you're looking for, bear in mind that a possible reason the tab isn't already on classtab.org is that i've been too busy answering your previous requests for help -- so before asking, search! and it may help to know some basic search query syntax (the use of brackets, quotes, plus & minus signs, Boolean ANDs & ORs etc)

thanks, in perhaps over-optimistic anticipation, and an acceptance that the majority of the world's population doesn't speak English -- so what i've just written will no doubt be ignored by many visitors :)


if you have the guitar tablature in PLAIN TEXT FORMAT for any of the following or if you know where they can be found, then please email me -- but before sending in tabs please check the latest version of this page to see if they are still wanted, and if possible play through them to make sure they are ok -- requests for sheet music are only for the purpose of contributing tabs to the Tabs Page -- wants may disappear after a year or so, but can be re-requested -- if you want to do a tab but can't find the sheet music, please ask

please do not send in unedited tabs created from MIDI files
